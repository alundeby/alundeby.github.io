Lighted A320 Models

This is the navigation light strobe sequence based on the actual Airbus A320 aircraft. Airbus uses a set pattern while most Boeing aircraft appear to have independent timing for the top, bottom, tail and wingtip strobes.

Electronics

First breadboard prototype of the idea using an Arduino Diecimila and TPIC6B595N shift register to control the LEDs.
Flashing the code to an ATTiny85 in order to save space and testing directly from a wall power supply.
I built a test rig using the same components as the ones that would be installed in the final model so that I had an easy way to test code changes.
Comparing 1206 and 0603 surface mount LED size options for the wingtip lights. Of course the smaller ones look better and are much harder to solder.
The 0603 LEDs are so small and light that it is impossible to solder them without some way to clamp them in place. I ended up grinding down a pair of tweezers to the exact width required to hold a pair of them back-to-back for the wingtip assemblies.
With the wingtip LED assemblies complete it was time to mount them on the wings. I cut a notch to help them sit flush with the leading edge of the wing and secured them with clear epoxy. I then ran the wires from each wingtip along the underside of the wing and secured it with epoxy as well.

Model Kit

Each fuselage contains a zinc weight in order to keep the plane resting on its nose gear. I needed to figure out the minimum size hole that I had to cut to remove it, which required sacrificing a kit.
Once I had the dimensions for the hole worked out I built a cutting jig. I used hot glue to create the actual clamp that holds the fuselage in place. It's safe on the paint and has just enough flex when removed from the MDF part of the jig that it's easy to insert and remove the fuselage.
My cutting tool was an adjustable speed Dremel with a router attachment so that I could set the cutting depth just above the zinc weight.

3D Insert

I designed a 3D printed part to replace the missing section of fuselage. The cutout was designed to hold a barrel jack which would also serve as the mounting point for the finished aircraft.

The first version of the insert had a tendency to get pushed into the fuselage near the tail. I created a second version that had a thin post that would keep that from happening by pushing against the top of the plane from the inside. I also added slots to completely surround the wing tabs and lock the entire assembly in place once they were inserted.

Assembly

There is not a lot of space to work with inside of these models. I had to split the ATTiny and shift register as neither end had enough space to hold both ICs with the wing tabs inserted.
The finished models on 3D-printed display stands.
